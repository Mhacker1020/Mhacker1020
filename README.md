## Tools

<table>

<td width="50%">

### [gate-cli](https://github.com/Mhacker1020/gate)
Supply chain security scanner. Detects malicious, typosquatted, and suspicious packages before you install them.

```bash
pip install gate-cli
gate scan requirements.txt
```

![Python](https://img.shields.io/badge/Python-3.12+-blue?logo=python&logoColor=white)
![PyPI](https://img.shields.io/pypi/dm/gate-cli?label=downloads)

</td>
<td width="50%">

### [zipguard](https://github.com/Mhacker1020/zipguard)
Security-focused archive extraction CLI. Prevents Zip Slip, archive bombs, symlink abuse, executable drops, and ZIP64 manipulation — before writing anything to disk.

```bash
pip install zipguard
zipguard archive.zip --out ./output
```

![Python](https://img.shields.io/badge/Python-3.12+-blue?logo=python&logoColor=white)
